### 들어가며
git push를 했을때 가끔 gitignore 설정이 제대로 안될때가 있다. <br>
git cache 를 지우고 다시 commit push를 하면 제대로 작동을 한다.
  
```
git rm -r --cached .
git add .
git commit -m "Untracked files issue resolved to fix .gitignore"
```
